# Twitter-pictrueAUTOdownloading
  Twitter（現X） 自動圖片下載

## Description 描述
  在 Twitter（X）頁面自動捲動，偵測推文圖片載入，並自動批次下載圖片檔。
  
## Supported this Types 程式支援類型
  瀏覽器：Chrome /IE /Brave
  需要登入Twitter（X）帳號
  
## Feature 主要功能
  1. 自動視窗捲動更新
  2. 偵測圖片數量
  3. 同步自動下載
  4. 簡易操作頁面

## Getting Started 安裝教學
  1. 打開 Chrome /IE /Brave 瀏覽器，進入 擴充功能管理頁面。
  2. 打開 開發者模式。
  3. 點擊 加載已解壓縮的擴充功能。
  4. 選擇包含 manifest.json 的目錄。
  
## How to Use 如何使用
  1. 開啟 X 頁面，並登入帳戶。
  2. 開啟擴充功能，會顯示彈跳頁面。
  3. 可F12 開啟Console 頁面查看運行過程。
