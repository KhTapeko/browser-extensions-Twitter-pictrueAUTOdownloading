# Twitter-pictrueAUTOdownloading
  Twitter（現X） 自動圖片下載

## Description 描述
  在 Twitter（X）頁面自動捲動，偵測推文圖片載入，並自動批次下載圖片檔。
  
## Supported this Types 程式支援類型
  瀏覽器：Chrome /IE /Brave
  需要登入Twitter（X）帳號
  
## Feature 主要功能
  1. 自動視窗捲動更新
  2. 偵測圖片數量
  3. 下載
  4. 簡易操作頁面

## Getting Started 安裝教學
  無
  
## How to Use 如何使用
  1. 開啟 X 頁面
  2. F12 開啟 DevTools → Console
  3. 貼上整段腳本 → Enter
